## 클래스 바인딩
- 객체 바인딩
  디렉티브를 사용한 클래스 바인딩의 경우 일반 class 속성과 공존할 수 있음
  객체를 반환하는 computed에 바인딩할 수 있음

- 배열에 바인딩
  배열에 :class를 바인딩하여 클래스 목록을 적용할 수 있음


## Style 바인딩
: html style 속성에 객체를 바인딩 할 수 있음
 > 객체에 직접 바인딩 하는 것이 좋음

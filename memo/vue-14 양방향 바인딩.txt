### 양방향 바인딩 [https://www.gymcoding.co/37ecf6bc-80e1-4d5e-ba2a-94e32804d931/]
: 입력 요소의 상태와 js의 상태를 동기화해야하는 경우
  v-model 디렉티브 사용
  배열을 바인딩하는것도 가능

실제 사용 시 단방향 바인딩과 이벤트 핸들러로 구현해야하는 경우가 있음
html 요소별로 구현방법이 다르므로 양방향 바인딩, 단방향 바인딩+이벤트 구현 방법 모두 숙지하는 것이 좋음

* 기본적으로 v-model 디렉티브는 각 input이벤트 후 입력과 데이터를 동기화 함

## v-model의 수식어

  .lazy = change이벤트 이후 동기화(포커스를 잃었을 때)
  .number = 사용자 입력을 자동으로 number 타입으로 형변환(기본 문자열 타입)
  .trim = 사용자 입력 내용에서 자동으로 앞뒤 공백을 제거하는 트림처리(v-model이 관리하는 input)

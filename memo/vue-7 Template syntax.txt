### 템플릿 문법(Template Syntax) [https://www.gymcoding.co/cd0b4825-9bcb-4c00-9697-e480137c120e/]

## 텍스트 보간법
: 데이터 바인딩의 가장 기본형태는 {{ data }} 를 사용하는 것임

* v-once 디렉티브 사용으로 데이터가 변경되어도 갱신(반응)되지 않는 일회성 보간을 수행할 수 있음



## HTML(v-html)
: 이중 중괄호는 데이터를 html이 아닌 일반 텍스트로 해석함. 실제 html을 출력하기 위해서는 v-html 디렉티브 사용
  *XSS 취약점


## 속성 바인딩(v-bind)
: 이중 중괄호는 html 속성에 사용할 수 없음, 대신 v-bind 디렉티브를 사용 / 단축표현 ':' 사용 가능
  다중 속성 바인딩으로 여러 속성을 한 번에 바인딩 할 수 있음

### 컴포넌트의 이해 [https://www.gymcoding.co/43211f50-c167-4f45-9582-625dd7c5cf40/]


* 모듈이란?
: js코드를 재사용할 수 있도로 분리한 파일


* 컴포넌트란?
: UI를 재사용할 수 있도록 정의한 것
  html, css 까지 캡슐화 하여 재사용 가능
  Vue는 넓은 의미에서 모든 인스턴스가 컴포넌트임

* 컴포넌트 사용 이유
 1. 재사용성
 2. 가독성 향상


## 컴포넌트 만들기
1. 컴포넌트 정의 -> 컴포넌트 등록 -> 컴포넌트 사용


# 컴포넌트 정의
: 정의 방법에 따라 두 가지 방법이 있음
    - 문자열 템플릿
    - SFC(*.vue)

# 컴포넌트 등록
: Vue 인스턴스의 component 메서드를 사용하여 컴포넌트 등록

# 컴포넌트 사용

* Vue.createApp 메서드의 파라미터는 ROOT 컴포넌트를 생성할 때 사용하는 옵션임
* SFC 컴포넌트로 개발을 하기 위해 Vite, Vue CLI와 같은 build tool이 필요 (npm install)

### Vue란 무엇인가? [https://www.gymcoding.co/eb73a52b-f54a-48da-a4ab-b22b7f499a1c/]
: user interface 개발을 위한 javascript frame work


## Vue 핵싱기능
- 선언적 렌더링(Declarative Rendering) : 템플릿 구문({{ data}})을 활용하여 데이터를 선언적으로 출력(렌더링)할 수 있음
- 반응성(Reactivity) : js상태 변경을 자동으로 추적, 변경이 발생하면 DOM을 효율적으로 업데이트


* v- 접두어가 붙은 특수 속성을 디렉티브(directive)라고 함

# 바인딩(v-bind)
: 템플릿 구문 이외 엘리먼트 속성에 데이터를 바인딩 하는 것, 반응형 동작

# 양방향 바인딩(v-model)
: 사용자 입력으로 DOM의 속성이 변경됐을 때 상태값도 함께 변경할 수 있도록 바인딩 하는 것

# 이벤트핸들링(v-on)
: Vue 인스턴스의 메서드를 호출하는 이벤트 리스너 추가 가능

### Template refs [https://www.gymcoding.co/e04d30f9-6ede-460f-a128-50a52f542b1e/]
: vue의 선언적 렌더링 모델은 대부분 직접적인 DOM의 작업을 대신 수행,
  하지만 기본 DOM요소에 직접 접근해야 하는 경우 ref 특수 속성을 이용해 쉽게 접근 가능


## Refs 접근
: 
  - <template> 안에서 input으로 refs참조에 접근, 렌더링 이전에는 null
  - <template> 안에서 $refs 내장 객체로 Refs참조에 접근
  - v-for 내부에서 ref가 사용될 떄 마운트 후 요소 배열로 채워짐

# Functions Refs
: ref속성에 문자열 대신 함수 바인딩 가능


## 컴포넌트 Refs
: ref를 자식 컴포넌트에서 사용 가능. 
  ref로 자식 컴포넌트의 참조값을 얻게 되면 자식 컴포넌트의 모든 속성과 메서드에 대한 전체 접근 가능
   > 단, 이러한 경우 부모-자식 간 의존도가 생기기 때문에 반드시 필요한 경우에만 사용/props를 사용하여 상호작용 하는 것이 일반적
   > 자식 컴포넌트에서 상위 컴포넌트를 참조하기 위해서는 $parent 내장객체 사용
